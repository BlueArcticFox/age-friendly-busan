# age-friendly-busan
## 활기찬 노후, 일하는 노후, 세대가 통합되는 고령친화도시, 부산
   
> ### 고령친화도시 조성의 필요성
> - 급속한 인구 고령화
> - 베이비부머 세대의 본격 은퇴 및 노인인구의 증가에 대비
> - 모든 시민이 체감하는 노인복지계획 필요
